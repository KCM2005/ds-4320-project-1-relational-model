I chose to use line graphs across age groups per insulin usage category for three reasons: (a) To visualize trends in insulin usage over a continuous variable. In this case, "age" is treated as a continuous variable. By plotting insulin usage along the y-axis and age groups along the x-axis, I am able to clearly see if insulin usage generally increases, decreases, or stays consistent as individuals get older within each category of insulin usage. (b) To compare insulin usage categories. By plotting separate line graphs for each "insulin usage category," I am able to directly compare how the age-related trends differ between these categories such as if a particular insulin regimen shows a steep increase in usage among older adults, while another shows a more stable usage pattern. (c ) To identify specific age brackets where insulin usage might peak or drop significantly for certain categories, ultimately leading to hypotheses about the underlying physiological reasons or treatment adjustments needed at those ages.
