The analysis rationale involves several key decisions. Logistic regression was chosen for its suitability in predicting binary outcomes (in this case, readmitted status was either yes/no) and providing interpretable coefficients for insulin usage patterns. The insulin usage categories ("Down", "No", "Steady", "Up") were encoded as categorical features to capture their impact on readmission risk relative to a reference category. Therefore, the logistic regression model assesses how these insulin usage categories, alongside other potential covariates (such as age) influence 30-day readmission likelihood. This approach allows for identifying high-risk groups and informing targeted interventions. Model performance was evaluated using precision accuracy.
